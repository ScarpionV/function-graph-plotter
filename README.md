# Function Graph Plotter 📊

A modern Python GUI for plotting mathematical function graphs, built with `customtkinter`. It features interactive plots, an intuitive user interface, and a clean dark-mode aesthetic.

This desktop application takes mathematical expressions and X-axis ranges (Start and End values) directly from the user to dynamically generate and display 2D graphs within the interface.

## 🚀 Features

* **Modern UI:** A sleek, user-friendly interface designed with `customtkinter`, fully optimized for a dark theme.
* **Dynamic Plotting:** Evaluates mathematical expressions (e.g., `x**2`, `np.sin(x)`) on the fly and renders them using `matplotlib`.
* **Flexible Bounds:** Users can freely define custom start and end intervals for the X-axis.
